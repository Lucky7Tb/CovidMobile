# CovidMobile
Aplikasi mobile, Tentang berita corona secara realtime

# Instalasi

### Clone Project
`git clone https://github.com/Lucky7Tb/CovidMobile.git`

### Install dependency
`npm install`

### Run Project
`/path/to/project>npx react-native run-android`
atau
`/path/to/project>react-native run-android`
